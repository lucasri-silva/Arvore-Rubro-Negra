# Arvore-Rubro-Negra

<h2>Projeto consiste na implementação da Árvore Rubro-Negra e na sua comparação com as árvores de pesquisa binária (comum) e AVL.</h2>

- Compilador
                
       gcc version 9.3.0
- OS
       
       Ubuntu 20.04.2 LTS

- Executar
 
        terminal:
        ./make clean
        ./make
        ./make run
